Description
Home page To access home page: https://fhakim-cloudflare-demo.com/
Message page Message page will display the result of a GET request for a webservice that will return the text value of 'msg' variable in the request, for example: https://fhakim-cloudflare-demo.com/getmessage?msg=Welcome to my demo
Users page 'Users page' path is secured by cloudflare through lock down access for only authenticated users and will display the result of GET request to return all the users from USERS JSON object: https://fhakim-cloudflare-demo.com/getusers
'/getheaders' post request a post request to '/getheaders' will return all headers in the request, including any added ones, for example:
curl --location --request POST 'https://fhakim-cloudflare-demo.com/getheaders' \
--header 'name: fhakim-cloudflare'
Test curl request #1 Test cloudflare worker script that redirect to "https://developers.cloudflare.com/" for any CURL requests for 'https://fhakim-cloudflare-demo.com/' : for example: curl -I 'https://fhakim-cloudflare-demo.com/' (the response header will have the redirect URL and status code 302)
Test curl request #2 Test cloudflare worker script that won't redirect based on CURL request with "cf-noredir=yes" cookie in the request for 'https://fhakim-cloudflare-demo.com/': for example: curl -I --cookie "cf-noredir=yes" 'https://fhakim-cloudflare-demo.com/' (in the response header, no redirect and status code 200)
